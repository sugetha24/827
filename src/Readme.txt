<<Readme>>
The compiler program contains:
1. Lexical analyser
2. Parser
3. Semantics
4. Pragmatics

To run the program:
1. Change the input file name in the mypro1.py file
2. run the mypro1.py
3. A .asm file will be created in the same directory
4. Open the terminal
5. Type the following command:
	$ ./run <filename>
	
	Eg:
	$ ./run sorting
6. The output will be geberated.
